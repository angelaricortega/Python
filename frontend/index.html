Conectar Google Forms con la API
Cada vez que alguien envíe tu formulario, la respuesta llegará automáticamente a esta API mediante Google Apps Script.
POST /encuestas/google-forms/
🪜 Pasos de configuración
1
Crear el formulario
En Google Forms, crea las preguntas con los títulos exactos de la tabla de abajo.
2
Abrir Apps Script
En el formulario: ⋮ → Editor de scripts. Borra el contenido y pega el código.
3
Cambiar la URL
Reemplaza API_URL con la URL de tu servidor (necesitas ngrok o nube).
4
Configurar trigger
Ícono del reloj → + Agregar activador → onFormSubmit → Al enviar formulario.
5
Probar
Ejecuta probarWebhook() en el editor o usa el botón de abajo.
