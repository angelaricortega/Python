¿Qué es el Índice NPL?

Fórmula

NPL = C + D + ETotal Cartera

Midela proporción de cartera en mora (categorías C, D, E) respecto al total

Clasificación Regulatoria

A Normal Cartera sana
B Observación Alerta temprana
C Subestándar Mora 1-30 días
D Dudosa Mora 31-90 días
E Pérdida Mora > 90 días

Umbrales de Riesgo

0% Sin Riesgo
<1% Riesgo Bajo
<5% Riesgo Moderado
<15% Riesgo Alto
≥15% Riesgo Crítico
¿Por qué es importante?

El índice NPL es el principal indicador de salud del sistema financiero. Un NPL alto indica que muchos préstamos no se están pagando, lo que puede llevar a crisis bancarias. La Superfinanciera colombiana usa este indicador para vigilar la estabilidad del sistema.

¿Qué es el Índice HHI?

Fórmula

HHI = ∑ (cartera_i)²(Total Cartera)²

Suma de los cuadrados de las carteras individuales dividido por el cuadrado del total

Interpretación

HHI < 0.15 Baja Concentración Cartera bien diversificada
0.15 - 0.25 Concentración Moderada Diversificación aceptable
HHI > 0.25 Alta Concentración Riesgo por falta de diversificación

¿Qué mide?

  • Concentración de riesgo: Qué tan distribuida está la cartera
  • Diversificación: Si el riesgo está en pocos o muchos municipios
  • Exposición máxima: El peso de cada municipio en el total
¿Cómo se interpreta?

El Índice HHI (Herfindahl-Hirschman Index) mide la concentración. Un HHI bajo (cercano a 0) indica que la cartera está bien distribuida entre muchos municipios. Un HHI alto (cercano a 1) significa que pocos municipios concentran la mayor parte del riesgo. Ejemplo: Si un solo municipio representa el 80% de la cartera total, el HHI será 0.64 (0.8²), indicando alta concentración.

¿Cómo calcula la API el Índice NPL?

NPL Ratio =
C + D + E Total Cartera

Solo C, D y E se consideran "cartera en mora". La API calcula automáticamente este y otros indicadores.

Formatos Numéricos Aceptados

Puedes ingresar los valores en cualquier formato. La API los interpreta correctamente.

1.500.000 Punto como miles (Colombia)
